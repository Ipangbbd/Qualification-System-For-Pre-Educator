## 🎨 Apple Design DNA

### Color Usage
- **Single accent**: `#0066cc` (Action Blue) - all interactive elements
- **Ink**: `#1d1d1f` - primary text
- **Tiles**: White → Parchment → Dark alternation
- No gradients, no decorative shadows (except product mockup)

### Typography
- **Hero**: 56px / 600 / SF Pro Display (tight tracking)
- **Body**: 17px (not 16px - Apple signature)
- **Negative letter-spacing** on all display sizes

### Interactions
- All buttons: `scale(0.95)` on active
- Pill-shaped CTAs (`border-radius: 9999px`)
- Backdrop blur on sub-nav (frosted glass)

### Product Showcase
- Dashboard mockup = MacBook-style presentation
- Single drop-shadow (Apple product treatment)
- Window chrome with macOS dots
- Live UI states (completed/active/locked)

---

## 📱 Responsive Behavior

- **Desktop (≥1024px)**: 3-4 column grids, dashboard full width
- **Tablet (641-1023px)**: 2 column grids, dashboard scales
- **Mobile (≤640px)**: Single column, dashboard stacks vertically

---

## 🚀 User Journey

```
1. Land on hero → See compelling headline
2. View dashboard → Understand what they'll use
3. Read benefits → See value for THEM
4. Learn process → Know exactly what to expect
5. Read testimonials → Build trust
6. Call to action → Easy signup with clear promises
```

**Not a sales page. An invitation to start their teaching career.**

---

## ✅ What Makes This User-Oriented

❌ **NOT focusing on:**
- Platform capabilities
- Admin features
- System architecture
- Company credentials

✅ **FOCUSING on:**
- User journey & experience
- Personal benefits & outcomes
- Transparency & trust
- Ease of use & accessibility
- Success stories from peers

---

## 🎯 Key Differentiators

1. **Dashboard as Hero Product** - Users see exactly what they'll interact with
2. **Visual Progress States** - Clear feedback at every stage
3. **User Testimonials** - Social proof from real teachers
4. **Time Promises** - "15 min to register, 3 days to results"
5. **No Hidden Info** - Upfront about retry policy, duration, process

This is not selling a platform. This is **inviting users to achieve their teaching career goals** through a transparent, supportive qualification process.
